### 👋 소개

**24년간 프로덕션 소프트웨어를 직접 설계하고 출시해 온 솔루션 아키텍트**입니다. 데스크톱 애플리케이션, 백엔드 서비스, 클라우드 인프라, AI 파이프라인, 개발자 도구에 이르기까지 폭넓은 영역에서 실무 경험을 쌓아왔습니다.

저를 차별화하는 것은 **넓이 × 깊이** 조합입니다. 플랫폼을 단순히 *사용*하는 것이 아니라, 플랫폼을 **확장하는 도구**를 직접 만듭니다. AI 에이전트를 위한 MCP 서버를 만들고, 언어 경계를 넘어 문서 라이브러리를 포팅하고, 오픈소스 생태계를 위한 빌드 파이프라인을 자동화하고, 대부분의 사람들이 가능한지도 몰랐던 시절에 Windows Container 위에서 프로덕션 게임 서버를 운영했습니다.

> *"최고의 아키텍트는 여전히 코드를 직접 작성하는 사람이라고 믿습니다."*

### 💡 팀에 가져다주는 가치

* **코드를 쓰는 아키텍트** — 다이어그램만 그리지 않습니다. 프로덕션 시스템을 출시하고, NuGet 패키지를 발행하고, 실제 사용자가 있는 오픈소스 프로젝트를 유지보수합니다.
* **깊은 플랫폼 전문성** — .NET 생태계 24년, Azure와 AWS 양쪽에서의 실전 클라우드 아키텍처 경험.
* **AI 네이티브 사고** — AI API를 소비하는 것이 아니라, MCP 서버와 RAG 파이프라인을 직접 만듭니다.
* **커뮤니티 승수 효과** — 17년 MVP 수상 이력과 한국 최대 .NET 커뮤니티 운영이 증명하는, 주변 모두를 성장시키는 역량.
* **경계를 넘는 기술력** — C#, Python, 아키텍트급 기술 문서, 개발자 대상 문서화 모두에 능숙합니다.
